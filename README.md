Titanic survaival predictor -> ML algorithms Logistic Regression && Random Forest: predicts whether a passenger on the Titanic would survive or not, based on various characteristics
Web framework: Flask with Jinja2
Some unit tests.

The project uses the following technologies (based on the files in the repository):
Python - main programming language
Pandas - for data manipulation and analysis
NumPy - for scientific computing
Scikit-learn - for machine learning and preprocessing
Matplotlib/Seaborn - for data visualization
Flask, Jinja2 - Web Framework

Instructuions for use:
Installation and Setup

git clone https://github.com/yourusername/Titanic-Survival-Prediction.git
cd Titanic-Survival-Prediction
python -m venv venv
source venv/bin/activate  (За Windows: venv\Scripts\activate)

pip install -r requirements.txt          (soon will be created)

In the main folder, open bash then start the program with:  python app.py
